Use this tool to add staff members who were missed during payroll generation.

Method 1: Add Staff to Existing Period

Use this if you want to add a specific staff member (like Linus) to an existing payroll period.

1 Open your browser console (F12) and run these commands: 
// First, import the service
import { payrollService } from './lib/services';

// Find Linus's staff ID from the Staff Management page
// or run: const staff = await staffService.getAll('YOUR_ORG_ID');
const staffId = 'LINUS_STAFF_ID_HERE';

// Find the payroll period ID (from URL or Payroll page)
const periodId = 'PERIOD_ID_HERE';
const orgId = 'YOUR_ORG_ID_HERE';

// If the period is finalized, unfinalize it first
await payrollService.unfinalizePeriod(orgId, periodId);

// Add Linus to the payroll
const entry = await payrollService.addStaffToPayroll(orgId, periodId, staffId);
console.log('Entry created:', entry);

// Refinalize the period
await payrollService.finalizePeriod(orgId, periodId);

console.log('‚úÖ Done! Linus should now see this payroll period.');

Method 2: Regenerate All Entries

Use this if multiple staff members are missing. This deletes all entries and regenerates them.

1 Ensure all staff who should be included have staffStatus: 'Active'
2 Open browser console and run: 
import { payrollService } from './lib/services';

const periodId = 'PERIOD_ID_HERE';
const orgId = 'YOUR_ORG_ID_HERE';

// Unfinalize the period
await payrollService.unfinalizePeriod(orgId, periodId);

// Get all existing entries
const entries = await payrollService.getEntries(orgId, periodId);

// Delete all entries (manually delete from Firestore or use a loop)
// This step requires direct Firestore access

// Regenerate entries (will include all Active staff)
const newEntries = await payrollService.generateEntries(orgId, periodId);
console.log('Generated', newEntries.length, 'entries');

// Refinalize
await payrollService.finalizePeriod(orgId, periodId);

console.log('‚úÖ Done!');

Method 3: Check Staff Status First

Before doing anything, check Linus's status:

import { staffService } from './lib/services';

const orgId = 'YOUR_ORG_ID_HERE';
const allStaff = await staffService.getAll(orgId);

// Find Linus
const linus = allStaff.find(s => s.fullName.includes('Linus'));
console.log('Linus Status:', linus?.staffStatus);
console.log('Linus ID:', linus?.id);

// If status is not 'Active', update it:
if (linus && linus.staffStatus !== 'Active') {
    await staffService.update(linus.id, { staffStatus: 'Active' });
    console.log('‚úÖ Updated Linus to Active');
}
